# Dramatic Prompts Library

This directory contains domain-specific prompts for the serialized dramatic verse layer in Sourceverse / Groetry.

- dramatic-verse-base.json → General fallback
- political-dramatic-voice.json → Strong Blakean political satire
- scientific-dramatic-voice.json → New: Cosmic / discovery themes
- technological-dramatic-voice.json → New: AI, machines, hubris
- cultural-dramatic-voice.json → New: Spectacle, memory, ritual
- philosophical-dramatic-voice.json → New: Eternal questions
- artistic-dramatic-voice.json → New: Creation and beauty

vertical_thread3.js now selects the appropriate voice based on domain detection.
